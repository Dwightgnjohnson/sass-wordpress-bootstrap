About This Thing

This package includes everything you need to get started on a WordPress project with Sass and Bootstrap. Its all open source, hack as you please.

The Sass part of things is powered by Sass Twitter Bootstrap, a 1:1 Less to Sass conversion of Bootstrap.

The WordPress theme functionality is powered by WordPress Bootstrap which was built on top of Bones.

Documentation can be found for Sass, Bootstrap, and WordPress.
